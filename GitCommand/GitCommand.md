# Gitコマンド 
##### origin : リモートリポジトリの場所(URL)にデフォルトでつけられる名前
##### main : メインブランチ名
##### origin main : リモートリポジトリのメインブランチ
##### origin/main : origin main を追跡するリモート追跡ブランチ

### リモートから変更を取得
```
git pull origin main
```
または
```
git fetch origin main
git merge origin/main
```
